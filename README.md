# Linux System Inventory

A Bash-based system inventory script for Linux workstations and servers.

## Purpose

This script collects useful system information for troubleshooting, documentation, audits and migration planning.

## Features

- System information
- CPU and memory details
- Disk and filesystem layout
- Network information
- Installed packages
- Running services
- Third-party software indicators
- Basic hardware overview

## Usage

```bash
chmod +x system-inventory.sh
./system-inventory.sh
